# 🧪 ml-experiments

> Сборник учебных репозиториев по машинному обучению — от классических алгоритмов до дообучения трансформеров.

---

## 📖 О сборнике

Этот репозиторий — личное учебное пространство для экспериментов с ML. Здесь собраны проекты, которые помогли мне разобраться в базовых и не очень базовых концепциях машинного обучения: от реализации линейной регрессии с нуля до файнтюнинга BERT на русскоязычных новостях.

**Предыстория.** Все эти репозитории существовали ещё до рефакторинга моего GitHub. Когда пришло время навести порядок, я решил не смешивать учебные эксперименты с портфолио-проектами — и собрал их в отдельный сборник. Здесь нет продакшн-кода и нет цели впечатлить: только честное обучение и попытки разобраться, как работают модели изнутри.

## 🗂️ Проекты

### 1. [`linear-regression-from-scratch`](./-linear-regression-from-scratch/)

Реализация линейной регрессии **без каких-либо ML-фреймворков** — только NumPy и математика.

Вручную реализованы: инициализация весов, градиентный спуск, расчёт метрик MSE и MAPE. Используется синтетический датасет с чёткой линейной зависимостью, чтобы модель можно было контролировать и визуализировать без лишнего шума. К последним итерациям модель достигает MAPE ~5.3%.

**Стек:** Python, NumPy

---

### 2. [`Logistic-Regression-on-Breast-Cancer-Dataset`](./Logistic-Regression-on-Breast-Cancer-Dataset/)

Классификация опухолей (злокачественная / доброкачественная) на датасете `breast_cancer` из `sklearn.datasets`.

Проект покрывает полный базовый пайплайн: нормализация данных, разбивка на train/test, обучение логистической регрессии, оценка точности и 5-фолдовая кросс-валидация. Датасет содержит 569 образцов и 30 числовых признаков.

**Стек:** Python, scikit-learn

---

### 3. [`Breast-Cancer-Predictor-Bot`](./Breast-Cancer-Predictor-Bot/)

Telegram-бот, построенный поверх модели из предыдущего проекта. Принимает от пользователя 30 медицинских параметров и выдаёт предсказание о характере опухоли.

Логическое продолжение `Logistic-Regression-on-Breast-Cancer-Dataset`: та же модель, но обёрнутая в интерактивный интерфейс. Проект сделан исключительно в образовательных целях — не для реальной диагностики.

**Стек:** Python, scikit-learn, pyTelegramBotAPI

---

### 4. [`catboost-in-credit-scoring-problem`](./catboost-in-credit-scoring-problem/)

Задача кредитного скоринга: предсказать, будет ли одобрен кредит на основе анкеты с 18 признаками (690 объектов).

Включает развёрнутый EDA: распределения, боксплоты, матрица корреляций. Модель — `CatBoostClassifier` с ручной настройкой гиперпараметров (depth, learning_rate, iterations). ROC-AUC на тесте — **0.957**. Хороший пример того, почему CatBoost удобен для работы с категориальными признаками без OHE.

**Стек:** Python, CatBoost, pandas, matplotlib

---

### 5. [`Telco-Customer-Churn-Prediction`](./Telco-Customer-Churn-Prediction/)

Предсказание оттока клиентов телеком-компании на реальном датасете с Kaggle (7043 объекта, 21 признак).

Особенность проекта — осознанный выбор метрики: акцент на Recall, так как пропущенный отток опаснее ложного срабатывания. Отбор признаков, подбор порога по F1, визуализация ROC-кривой и важности признаков. Итоговая точность — **0.80**, macro F1 — **0.76**.

**Стек:** Python, CatBoost, scikit-learn, matplotlib

---

### 6. [`nlp-faq-model`](./nlp-faq-model/)

Telegram-бот для теоретического магазина, отвечающий на FAQ с помощью семантического поиска.

Вопрос пользователя сравнивается с базой знаний через косинусное сходство эмбеддингов модели `paraphrase-multilingual-MiniLM-L12-v2`. Если схожесть выше порога — бот возвращает готовый ответ, иначе просит переформулировать. Первый контакт с NLP и sentence-transformers.

**Стек:** Python, sentence-transformers, pyTelegramBotAPI, loguru

---

### 7. [`sneakerFAQ`](./sneakerFAQ/)

Расширение `nlp-faq-model`: теперь бот умеет отвечать не только на общие вопросы магазина, но и на вопросы о конкретных кроссовках из каталога.

Данные о товарах хранятся в SQLite, продуктовые тексты — в JSON-файлах. При первом запуске модель скачивается автоматически. Проект упакован в Docker.

**Стек:** Python, sentence-transformers, SQLite, Docker, pyTelegramBotAPI

---

### 8. [`transformer-news-finetune`](./transformer-news-finetune/)

Дообучение `BertForSequenceClassification` на основе `DeepPavlov/rubert-base-cased` для классификации новостных заголовков с Lenta.ru по 3 категориям.

Включает скрипт парсинга данных, настройку Hugging Face `Trainer`, логирование обучения. Финальные метрики: accuracy **0.9895**, macro F1 **0.9903**. Есть отдельная ветка с веб-версией на FastAPI + Streamlit.

**Стек:** Python, Hugging Face Transformers, PyTorch, FastAPI, Streamlit
