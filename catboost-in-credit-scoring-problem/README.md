# 💳 CatBoost для кредитного скоринга

Простой и наглядный пример применения алгоритма **CatBoost** на задаче **кредитного скоринга**.

Мы предсказываем, будет ли клиенту **одобрен кредит** (**1** — одобрено, **0** — отклонено) на основе анкеты с 18 признаками.

## ⚙️ Предобработка

Ключевые шаги:

* **Масштабирование** числовых признаков через `MinMaxScaler`.
* **Автоматическое определение** категориальных признаков по типу данных.
* **Разделение** на обучающую и тестовую выборки с сохранением пропорций классов (`stratify=y`).

---

## 🤖 Модель: CatBoostClassifier

| Параметр | Значение | Описание |
| :--- | :--- | :--- |
| **Алгоритм** | `CatBoostClassifier` | Оптимизирован для категориальных признаков. |
| `iterations` | `500` | Количество деревьев (итераций обучения). |
| `depth` | `6` | Максимальная глубина деревьев. |
| `learning_rate` | `0.05` | Скорость обучения. |
| `loss_function` | `"Logloss"` | Функция потерь для бинарной классификации. |
| `eval_metric` | `"AUC"` | Метрика для отслеживания на валидационной выборке. |
| `random_seed` | `42` | Для воспроизводимости результатов. |

> **Примечание:** Используется объект `Pool` для `CatBoost`, чтобы обеспечить правильную обработку категориальных признаков без предварительного One-Hot Encoding.

## 📈 Метрики и графики

### Метрика ROC-AUC

* **ROC-AUC: 0.957** — это **высокая** способность модели различать одобренные и отклонённые кредиты.

### ROC-кривая

Визуализирует **TPR** (чувствительность) против **FPR** (1 - специфичность) при различных порогах. Показывает компромисс между этими двумя показателями.

```python
from sklearn.metrics import roc_curve, auc
import matplotlib.pyplot as plt

y_proba = model.predict_proba(X_test)[:, 1]
fpr, tpr, thresholds = roc_curve(y_test, y_proba)
roc_auc = auc(fpr, tpr)

plt.figure(figsize=(8, 6))
plt.plot(fpr, tpr, color='blue', lw=2, label=f'ROC-кривая (AUC = {roc_auc:.3f})')
plt.plot([0, 1], [0, 1], color='red', lw=2, linestyle='--', label='Случайная модель')
plt.xlabel('Доля ложноположительных (FPR)')
plt.ylabel('Доля истинноположительных (TPR)')
plt.title('ROC-кривая')
plt.legend(loc='lower right')
plt.grid(True)
plt.show()
